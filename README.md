## 各ゴールと推奨ツール

### 【A】改善余地の洗い出しへ  
- 目的：既存施策のボトルネックを特定  
- 推奨ツール：Notion＋AI分析、Googleアナリティクス、アンケートAI要約

### 【B】LINE公式アカウントの自動化構築  
- 目的：相談前の教育と予約導線の自動化  
- 推奨ツール：LINE公式アカウント、ChatGPTで配信スクリプト、Zapier/Make

### 【C】口コミ・再来導線強化  
- 目的：成約後の口コミ獲得と再訪促進  
- 推奨ツール：Googleビジネスプロフィール、テンプレAI作成、LINEセグメント配信

### 【D】さらなるブランドの拡大（動画・SEO）  
- 目的：専門性発信で地域No.1の認知へ  
- 推奨ツール：YouTubeショート/TikTok、note/ブログ、AI台本生成

### 【E】動画・SNS露出強化  
- 目的：まずは知ってもらう入口を増やす  
- 推奨ツール：Canva＋Magic Design、Descript/Vrew、ChatGPTでテーマ抽出

### 【F】検索導線の整備（Googleビジネス・ブログ）  
- 目的：検索からの信頼獲得と予約導線  
- 推奨ツール：Googleビジネスプロフィール、note、AIライティング支援

### 【G】セミナー・教育コンテンツで信頼醸成  
- 目的：リアルタイムの信頼→相談へ  
- 推奨ツール：Zoomウェビナー、録画→YouTube再利用、AI資料設計

### 【H】高度なAI活用で効率化  
- 目的：業務省力化とデータ活用  
- 推奨ツール：ChatGPT活用テンプレ、CRM＋AI要約、Zapier自動連携

### 【I】既存顧客向け情報発信とAI要約  
- 目的：既存顧客の満足度・紹介率アップ  
- 推奨ツール：LINEオープンチャット、ニュース要約AI、定期note配信
